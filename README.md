# Useful Stone

**Useful Stone** is a small quality-of-life mod that fixes minor inconsistencies in Minecraft's crafting system.
In vanilla Minecraft, blocks like furnaces can be crafted using different stone variants such as **deepslate** or **blackstone**, but many other stone-based recipes only accept cobblestone. This mod improves that

The mod is lightweight, vanilla-friendly, and designed to keep Minecraft's crafting more consistent.

---

### Features
- Allows crafting certain blocks with **cobblestone, deepslate, or blackstone**
- Improves consistency in Minecraft's crafting recipes
- Affects blocks such as:
  - Lever
  - Dispenser
  - Dropper
  - Observer
  - Piston
- Works on **Fabric and Forge**
